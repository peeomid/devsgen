# CodeTweak - Project Overview

**CodeTweak** is a collection of developer utilities designed to streamline common tasks and boost productivity. It's built as a modern web application with a focus on client-side processing, keyboard shortcuts, and developer-friendly workflows.

## Project Purpose
- Provides developer tools like regex helpers with patterns for common transformations
- Client-side processing - data never leaves the browser for privacy
- Keyboard-driven workflows with minimal mouse usage
- SEO-optimized with dedicated pages for each tool and pattern

## Key Features
- **Regex Helper**: Transform text using regex patterns with built-in and custom patterns
- **Quick Access**: Numeric key access and short key lookup for patterns
- **Pattern Management**: Create, edit, delete, and search regex patterns
- **Command Palette**: Keyboard-driven pattern selection (Cmd+K)
- **Import/Export**: JSON-based pattern sharing
