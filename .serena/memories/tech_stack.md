# CodeTweak - Technology Stack

## Core Technologies
- **Astro 5.5.2**: Static site generator with islands architecture, SSR enabled
- **React 18.2.0**: Interactive UI components
- **TypeScript 5.4.2**: Type-safe development with strict configuration
- **Tailwind CSS 3.4.1**: Utility-first CSS framework
- **Nanostores 0.10.0**: Lightweight state management

## Development Tools
- **ESLint 8.57.0**: Code linting with TypeScript and React plugins
- **Vitest 1.4.0**: Unit testing framework
- **nanoid**: Unique ID generation for patterns

## Configuration
- TypeScript config extends `astro/tsconfigs/strict`
- Astro configured with React and Tailwind integrations
- SSR output mode for better SEO
- Site URL: https://CodeTweak.osimify.com
